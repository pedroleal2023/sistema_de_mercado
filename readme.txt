# Sistema de Mercado 🛒

Este é um sistema simples para gerenciamento de um mercado, desenvolvido com o objetivo de praticar habilidades de programação e estruturação de projetos.  

## 📋 Funcionalidades

- Cadastro de produtos com nome, preço e quantidade.
- Gerenciamento de estoque.
- Simulação de vendas, com atualização automática do estoque.
- Cálculo do total de vendas realizadas.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Java
- **IDE:** IntelliJ IDEA / Visual Studio Code
- **Gerenciamento de Dependências:** Maven/Gradle (caso aplicável)
